## ‚öôÔ∏è Create Your Own Template

If you prefer to create a custom template, click the **"Create Template"** button, which will open a new page for you to define your template. Here's how it works:

### 1. Template Details

- **Template Title**: Give your template a title.
- **Template Description**: Provide a description to explain what your template does.
- **Template Icon**: Paste the SVG code for your desired icon (from Tabler Icons or any other icon set).
- **Template Color**: Choose a color for your template (e.g., `#8fd2d0`).

### 2. Input Groups

Define the input fields for your template:

- **Select Input Type**: Choose between different types of input (e.g., Text, Number, Dropdown).
- **Input Name**: Enter the name for the input field (e.g., "Brand Name").
- **Input Description**: Describe what the input field is for (e.g., "Enter your brand's name").

### 3. Custom Prompt

Define a custom prompt that will guide the AI in generating content based on the user's inputs:

- **Custom Prompt**: Write the prompt the AI will use to generate content (e.g., "Create a catchy social media post for [Brand Name]").

Click **Save** to store your custom template and make it available for future use.

## ‚ö° AI Writer Interface

Once you've selected a template (or created your own), the AI Writer opens with a split-screen interface:

- **Left Side**: This is where you input information such as:
  - **Include Your Brand**: Add brand name or details.
  - **Generate Bulk Posts**: Option to generate multiple posts at once.
  - **Description**: A brief description for your content.
  - **Language**: Choose the language (e.g., English - USA).
  - **Maximum Length**: Define the maximum word count or character count for the generated content.
  - **Creativity**: Choose the creativity level (e.g., Low, Medium, High).
  - **Tone of Voice**: Specify the tone (e.g., Friendly, Professional, Persuasive).
  - **Select Model**: Choose the AI model for generation (e.g., GPT-4, Custom Model).

- **Right Side**: The AI-generated content will appear here. You can easily edit this content using the editor tools available in the **AI Editor**:
  - **Undo/Redo**
  - **Bold, Italics**
  - **Text Color, Background Color**
  - **Alignment (left, center, right)**
  - **Word Count**
  - **Assistant**: Access the AI assistant to help refine your content.

Once satisfied, you can generate the content by clicking the **"Generate"** button.
